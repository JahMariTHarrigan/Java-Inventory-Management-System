# Java-Inventory-Management-System
Java-based inventory management system that processes transaction files, updates stock records, flags low or out-of-stock items, and generates inventory reports for multiple job sites.

How to run: Must download the two txt files first then download stock.item, and then download and run the main StockReport.java
